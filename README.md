# Level Up Networking Question Bank

A static, free-to-host website for Jason P. Yoong's Level Up Networking Question Bank.

## What Is Included

- Branded landing section using Level Up colors and assets.
- Searchable/filterable question bank.
- List and card views.
- Copy-to-clipboard controls.
- Browser-local saved questions.
- Random prompt shuffle.
- Leaders Go First section.
- FAST virtual networking format guide.
- Clickable external resources page for Alisa Cohn, Shaan Puri, Tim Ferriss, and Arthur Aron.
- Original public Google Doc and LinkedIn DM links in the footer.

## Maintenance

The site is intentionally plain HTML/CSS/JS. To update questions, edit the `questions` array in `app.js`.

Recommended free hosting:

1. GitHub Pages for simplest free static hosting.
2. Netlify Drop for fastest no-repo launch.
3. Vercel if you later want a more app-like workflow.

Suggested domain names:

- `networkingquestionbank.com`
- `levelupnetworkingquestionbank.com`
- `levelupquestions.com`
